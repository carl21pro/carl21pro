## üöÄ How to Deploy on Vercel (Step-by-Step)

If you want to host this website for free using Vercel, follow these instructions:

1.  **Create a GitHub Repository:**
    - Log in to your GitHub account and create a new repository (e.g., `my-portfolio`).
    - Upload the three files (`index.html`, `style.css`, and `script.js`) to that repository.

2.  **Connect to Vercel:**
    - Go to [Vercel.com](https://vercel.com) and log in with your GitHub account.
    - Click the **"Add New"** button and select **"Project"**.
    - Find your repository in the list and click **"Import"**.

3.  **Deploy:**
    - Vercel will automatically detect the HTML/CSS/JS files.
    - Click **"Deploy"**.
    - Within 1 minute, your site will be live with a custom URL (e.g., `my-portfolio.vercel.app`).
